DETAILED DESCRIPTION BY FUNCTIONALITY (HUMAN PROTOCOL)
--------------------------------------------------------------------------------

────────────────────────────────────────────────────────────────────────────────
  OPERATING MODE & LIVE STATE (User mode)
────────────────────────────────────────────────────────────────────────────────
  mode 0          Set VOICE mode (voice filtering: HPF + LPF).
  mode 1          Set CW mode (bandpass filter around CW frequency).
  voice           Direct alias: enable VOICE mode (User live).
  cw              Direct alias: enable CW mode (User live).
  uservoice       Return to manual VOICE live control, leaving the active preset.
  usercw          Return to manual CW live control, leaving the active preset.
  When switching modes, live settings are automatically saved/loaded from NVS.

────────────────────────────────────────────────────────────────────────────────
  VOICE FILTER (mode 0, User live)
────────────────────────────────────────────────────────────────────────────────
  lowcut N        High-pass frequency (Low-cut) in Hz. Range: 25 – 800 Hz.
  hicut N         Low-pass frequency (Hi-cut) in Hz. Range: 1200 – 10500 Hz.
  slope N         Filter slope: 0=Soft, 1=Medium (24dB), 2=Sharp, 3=Bypass.

────────────────────────────────────────────────────────────────────────────────
  CW FILTER & SPATIAL PROCESSING (mode 1, User live)
────────────────────────────────────────────────────────────────────────────────
  cwcenter N      Center frequency in Hz. Range: 400 – 900 Hz.
  cwbw N          Bandwidth in Hz. Range: 25 – 1000 Hz.
  cw2don          Enable 2D spatial processing.
  cw2doff         Disable 2D spatial processing.
  cw2d N          Set delay (ms). Range: 1 – 25 ms.
  cw3don          Enable 3D processing.
  cw3doff         Disable 3D processing.
  cw3d N          Set 3D coefficient. Range: 1 – 10.
  cwagcon         Enable CW AGC.
  cwagcoff        Disable CW AGC.

  cwthr [N]       Detection threshold (%). Range: 1 – 100 (Default: 32).
  cwdelta [N]     Hysteresis margin. Range: 1 – 100 (Default: 32).

────────────────────────────────────────────────────────────────────────────────
  MP3 PLAYER CONTROLS & EFFECTS
────────────────────────────────────────────────────────────────────────────────
  mpvol N         Output volume (0–100 %).
  mpbal N         Balance (-100 left, 100 right).
  mp3drc          Enable/disable compressor.
  mp3thr N        Threshold (-40.0 … 0.0 dB).
  mp3gain N       Make-up gain (0.0 … 12.0 dB).
  mp3att N        Attack (1 – 100 ms).
  mp3rel N        Release (10 – 1000 ms).

────────────────────────────────────────────────────────────────────────────────
  MP3 PLAYER TRANSPORT - ACTION
────────────────────────────────────────────────────────────────────────────────
  play, stop, pause, next, prev, repeat, shuffle, mallsd, table, mcurtrack

────────────────────────────────────────────────────────────────────────────────
  AUDIO CONTROL - ACTION

────────────────────────────────────────────────────────────────────────────────

────────────────────────────────────────────────────────────────────────────────
  SYSTEM / DIAGNOSTICS - ACTION
